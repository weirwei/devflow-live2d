# Natori official demo model

This directory contains the `Natori` sample model copied from the official Live2D Cubism Web SDK package for local testing inside the desktop overlay.

Source:

- Package: `CubismSdkForWeb-5-r.4`
- Original path: `Samples/Resources/Natori`

Notes:

- `Natori` is used as the default runtime test profile because it includes named expressions such as `Smile`, `Surprised`, `Angry`, and `Sad`, which makes event-driven expression testing much easier than the simpler samples.
